# BlogAPI


Una API Rest que hace las opraciones basdicas (CRUD)
para un blog administrando sus publicaciones y los comentarios
de cada una de las publicaciones. 

-API REST Operaciones CRUD
-Pagination and Sorting
-Manejo de excepciones de la API REST
-Inicio de Sesión / Registro
-Validación de la API REST
-Protección de la API REST (seguridad basada en roles)
-Agregar capa de seguridad a la API REST usando JWT
-Pruebas de la API REST con Postman REST Client
